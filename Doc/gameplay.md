# 游戏玩法

## 第一层 南山悠然

### 拼诗 Ancient1

![](pic/poem_ancient1_1.png)

将散落的诗句以特定的顺序进行排序

![](pic/poem_ancient1_2.png)

正确匹配的诗句会显示绿色并停留在正确位置

### 拼画 Modern1

![](pic/paint_modern1_1.png)

将散落的碎片拼回画中

![](pic/paint_modern1_2.png)

正确匹配的碎片会停留在正确位置

### 引光 Future1

![](pic/light_future1_1.png)

将散落的镜子插入特定的凹槽

![](pic/light_future1_2.png)

镜子会改变光路，请将光路折射到目标上
